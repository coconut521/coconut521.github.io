根 Root

根的主要功能

具有固着和支持作用
吸收水、二氧化碳及溶解于水中的无机盐
储藏合成作用
其它：呼吸、繁殖, etc

根的形态

  • 定根
    • 主根 Main root
      由胚根生长出来的 ^6fd268
    • 侧根 Lateral root
      主根生长到一定程度就向后生长出许多侧根
  • 不定根 Adventitious root
    在根、茎、老根或胚轴上生出的根 ^59e736
  • Examples ^ff827b
    • 玉米的支持根
    • 萝卜的贮藏根
    • 榕树的气生根
    • 何首乌的块根
    • 红树林的呼吸根
    • 热带雨林中的板根

茎 Shoot

茎是植物生长在地上部的营养器官，其上着生叶、花和果实
茎通常是呈圆柱形，少数其它形状

^0f24d7

  • 茎包括芽、节、节间、枝条以及皮孔
  • 芽包括顶芽和侧芽（叶腋 内着生）
    • 节：叶片的着生处
    • 节间：两节之间的位置
    • 叶痕：叶子脱落后，叶柄 在茎上留下的痕迹
    • 皮孔：木质茎上交换气体的通道 （杨树的皮孔有特点）
  • 通常具有顶端优势

茎的主要功能

  • 输导作用
  • 支持作用：伸展枝、叶、花、果，利于光合作用、传粉和果实传播
  • 贮藏作用：鳞茎、块茎、球茎、根状茎等贮存大量养料
  • 繁殖作用：变态茎的自然营养繁殖；能产生不定芽和不定根，扦插、压条、嫁接等
  • 光合作用 e.g. 仙人掌

茎的形态分类

  • 直立茎
  • 匍匐茎（草莓）
  • 攀缘茎（爬山虎）
  • 缠绕茎（牵牛花）
  • 球茎（荸荠）
  • 根状茎（荷花）
  • 鳞茎（百合）
  • 块茎（土豆）
